# David's Comic Vault — Version 4.0

New in Version 4:
- Brief story summary for every photographed item
- Key events and key characters on each comic detail view
- Character index
- Browser-based issue verification editor
- Exportable correction JSON
- Existing Top 25, dashboard, publisher breakdown, box tracker, search, mobile support, dark theme, favorites and wishlist
- Original actual-cover filenames preserved
- No generated cover art

Corrections entered in the Verify Issues page are stored in the browser using localStorage.
